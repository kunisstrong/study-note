## linux命令

### 格式

命令 + 空格 + 参数（可写可不写）+ 空格 + 文件、文件夹（可写可不写）

### 名称含义

`wangkun@wangkundeMacBook-Air Desktop`

wangkun          @           wangkundeMacBook-Air         	Desktop 

登陆的用户   占位股			当前机器的名字					 当前所在工作目录

### linux命令

#### cd命令

更换目录

`cd  可选参数  文件夹`

几个特殊的目录

1. `.`：当前工作的目录
2. `..`：上一级工作的目录
3. `-`：上一次工作的目录
4. `~`：当前系统登录的用户家目录

#### ls命令

列出文件夹的内容

`ls  可选参数 可选的文件夹对象`

##### 常用参数

1. `-a`：参数。显示出文件夹所有文件内容，包括隐藏的
2. `-l`：详细输出文件夹内容
3. `-h`：以人类可阅读的形式，输出文件大小
4. `--full-time`：以完成的时间格式输出
5. `-t`：根据最后修改的时间排序文件

#### pwd命令

显示当前所有目录的绝对路径

print work directory  打印工作目录

#### vim命令

`vim 文件名 #打开文件`

##### 使用步骤

1. vim 文件名 打开文件

2. 输入字母i进入编辑模式，代表insert

3. 写完内容后，按下esc，退出编辑模式

   1. 输入`:`，进入底线命令模式，然后输出`wq`退出编辑

      `:wq!` 强制写入文件内容且退出vim,  write（写）、quit（停止）、!（强制）

### Linux目录结构

1. /dev： 存放抽象硬件
2. /lib：存放系统库文件
3. /sbin：存放特权级二进制文件
4. /var：存放经常变化的文件
5. /home：普通用户目录
6. /etc：存放配置文件目录
7. /boot：存放内核与启动文件
8. /bin：存放二进制文件（可执行命令）
9. /usr：存放安装程序（软件默认目录）
10. /mnt：文件挂载目录（u盘、光驱）
11. /root：特权用户目录
12. /opt：大型软件存放目录（非强制）
